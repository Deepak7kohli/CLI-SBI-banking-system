# CLI SBI Banking Management System

A console-based Banking Management System developed using **Core Java, JDBC, and MySQL**. This application simulates the basic functionalities of a banking system through a command-line interface (CLI).

---

## Features

- Create a new bank account
- Secure user login
- Deposit money
- Withdraw money
- Transfer money between accounts
- Check account balance
- View transaction history
- Change account password
- Update user profile
- Exit application safely

---

## Technologies Used

- Core Java
- Object-Oriented Programming (OOP)
- JDBC
- MySQL
- Eclipse IDE

## Database Setup

1. Install MySQL.
2. Create a database.
3. Execute the `database.sql` file.
4. Update the database username and password in the project.
5. Run the application.
